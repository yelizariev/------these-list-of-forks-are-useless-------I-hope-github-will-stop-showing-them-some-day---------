В зависимости от текущей установки существует несколько способов обновить версию сообщества. В любом случае основные рекомендации:

  • Резервное копирование базы данных сообщества

  • Выключите ваш сервер
  • Установите модуль web_enterprise
  • Перезагрузите ваш сервер
  • Введите код вашей подписки на Odoo Enterprise

В Linux, используя установщик

  • Резервное копирование базы данных сообщества

  • Остановить службу odoo

    $ sudo service odoo stop

  • Установите корпоративный .deb (он должен быть установлен поверх пакета сообщества)

    $ sudo dpkg -i <path_to_enterprise_deb>

  • Обновите базу данных до корпоративных пакетов, используя

    $ python3 /usr/bin/odoo-bin -d <database_name> -i web_enterprise --stop-after-init
  • Вы должны иметь возможность подключиться к своему экземпляру Odoo Enterprise, используя обычное средство идентификации. Затем вы можете связать свою базу данных с подпиской Odoo Enterprise, введя код, полученный по электронной почте, в форме ввода

В Linux, используя исходный код

Есть много способов запустить ваш сервер при использовании источников, и у вас, вероятно, есть свой любимый. Возможно, вам придется адаптировать разделы к вашему обычному рабочему процессу.

  • Выключите ваш сервер
  • Резервное копирование базы данных сообщества
  • Обновите параметр `` –addons-path`` вашей команды запуска (см . Источник Установить)

  • Установите модуль web_enterprise с помощью

    $ -d <database_name> -i web_enterprise --stop-after-init

    В зависимости от размера вашей базы данных, это может занять некоторое время.

  • Перезагрузите сервер с обновленным путем добавления аддонов из пункта 3. Вы должны иметь возможность подключиться к своему экземпляру. Затем вы можете связать свою базу данных с подпиской Odoo Enterprise, введя код, полученный по электронной почте, в форме ввода

На винде

  • Резервное копирование базы данных сообщества

  • Удалить Odoo Community (используя исполняемый файл Uninstall в папке установки) - PostgreSQL останется установленным

  • Запустите установщик Odoo Enterprise и выполните обычные действия. При выборе пути установки вы можете указать папку установки сообщества (эта папка все еще содержит установку PostgreSQL). Снимите флажок `` Start Odoo`` в конце установки

  • Используя командное окно, обновите базу данных Odoo, используя эту команду (из пути установки Odoo, в подпапке сервера)

    $ odoo.exe -d <database_name> -i web_enterprise --stop-after-init
  • Не нужно вручную запускать сервер, служба запущена. Вы должны иметь возможность подключиться к своему экземпляру Odoo Enterprise, используя обычное средство идентификации. Затем вы можете связать свою базу данных с подпиской Odoo Enterprise, введя код, полученный по электронной почте, в форме ввода
